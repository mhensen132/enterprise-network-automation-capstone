Project Overview

Objective:

Build a model-driven network automation framework using Cisco IOS-XE devices managed through Ansible and NETCONF.

Why This Project Matters:

Traditional CLI configuration does not scale well across multiple devices. NETCONF and YANG provide structured, programmable interfaces that support automation and consistency.
